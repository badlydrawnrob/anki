Recursion

``

sum : List Int -> Int
sum l =
    case l of
        [] -> 0
        x::xs -> x + (sum xs)
> sum [1, 2, 3, 4]
10 : Int

This is a recursive function that calculates the sum of the numbers passed to the function as a list.

We can use the :: operator to deconstruct the list (as well as constructing a list!) into it’s head and tail.
