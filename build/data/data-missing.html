What’s the difference between these two types?

Type

``

type Car =
    Car { name : String, age : Int }

type alias Car =
    { name : String, age : Int }
Car { name = "Audi", age = 10 }

type Car comes with NO CONSTRUCTOR function. It expects a single value, which in this case is a record. Our type alias Car allows us to construct a record with Car "Audi" 10.

It’s worth considering the difference as using Json.Decode.map will throw an error if you try to use it with type Car.
