Which function allows us to calculate this sum?

Maybe

Just 100 : Maybe number

num1 = Just 100
num2 = Just 50
empty = Nothing
> Maybe.map2 (/) num1 num2
Just 2
> Maybe.map2 (/) num1 empty
Nothing

Maybe.map “lifts” the value from it’s Maybe type, and .map2 takes a function for two values.

This function is very useful when we need a calculation, but don’t want to unwrap our Maybe types first. Remember, reaching for Maybe.withDefault should be the last thing you do. Here’s a more complex example:

values = [Just 100, Just 200, Just 100 ]

List.foldl
  (Maybe.map2 (+))
  (Just 0)
  values

false
