Accessing a dictionary item with it’s key returns it’s value. Similar to a list, but you access values by looking up a key instead of an index!

Dictionaries are enclosed in curly braces { }. You can change values (such as a string / number), or generate a brand new key/value pair:

# Change a value
residents['Burmese Python'] = 79
# Add an animal
residents['Giraffe'] = 200

Warning: Never search for a value in a dictionary using a for loop! The .items() method allows you to iterate over (key, values) as a tuple list. Unlike Elm Lang, Python values are mutable.
